<Update label="December 2025" description="v2026-01-01">
  **RPKI API Launch**
  - Added comprehensive RPKI (Route Public Key Infrastructure) API documentation
  - Route validation endpoints for checking BGP origin authorization
  - Bulk validation support for multiple routes in single request
  - Real-time delta updates and long-polling notification system
  - Infrastructure monitoring with trust anchor and repository status
  - Prometheus metrics integration for observability
  - 10 endpoints covering validation, monitoring, and delta updates

  **Certification & Badge API Launch**
  - Added complete Certification & Badge API documentation
  - New endpoints for certificate templates, digital badges, and credential management
  - Support for credential issuance, verification, and revocation
  - Webhook notifications and audit logging capabilities

  **API Versioning Updates**
  - Standardized header-based versioning across all APIs (`Hixbe-Version: 2026-01-01`)
  - Updated base URLs for all API endpoints
  - Enhanced error handling and response schemas
</Update>
